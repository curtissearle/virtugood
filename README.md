# VirtuGood 6500

Retro-future first-person experience built with React, Vite, and `react-three-fiber`.

## Controls

- `W/A/S/D` to move
- `Mouse` to look around
- `Space` to jump, `Ctrl` to crouch, `Shift` to walk slow
- `Click` to interact / select
- Press `Enter` on the overlay to start (locks the cursor)

## Getting Started

**Prerequisites:** Node.js 18+ and pnpm

1. Install dependencies  
   `pnpm install`

2. Start the dev server  
   `pnpm run dev`

3. Open the URL from the console (defaults to `http://localhost:3000`) and press `Enter` to lock the cursor.
